Aldon Chen for MICA Forum Fall 2020

Enjoy out of context objects? Me neither.

This website is an interactive art experience exploring value and perception in artwork. I conceived, designed, and developed this project in 2020 as a response to Stephen Hendee's project prompt "Camouflage" for MICA Forum 1.
As you explore this website, consider the following questions:
What is beauty/art?
How is beauty experienced? Is it seen? Felt? Or some other way?
How is the value of an art object determined?
Can something be considered art even if only a single individual considers it so?
